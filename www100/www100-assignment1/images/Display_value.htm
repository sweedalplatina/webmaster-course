Tutorial on using Display property

  • Check out the same tutorial with a different CSS

CSS stands for Cascading Style Sheets. It describes how the various HTML elements will be displayed. Layouts of multiple web pages can be controlled by a single CSS document. This tutorial is focused on the ‘Display’ property of CSS.

Display property states the way in which an element is arranged. Elements can be styled in various ways using the Display property. For most of the elements default display value is usually ‘Inline’ or ‘block’.

This tutorial would target the 3 commonly used values of Display property, namely:

  • Display: Inline
  • Display: Block
  • Display: Inline-block

Display property in CSS is a very helpful tool, may it be with any of its associated value.

Let’s go through each of these display values in details

Display: Inline

"Display: Inline" property on elements do not cause a line break. Once an element is displayed as an Inline element, it is impossible to add height and width to it.

Display: Block

Applying "Display: block" property to any element causes a line break and the element will be displayed as a block level element. Height and width can be given to these elements.

<span> tag causes a line break and acts like a block level element after being styled as "Display: block". Hence, the content in the <span> tag is displayed in the next line.

Display: Inline-block

Applying the "Display: Inline-block" property to an element makes it act like an Inline element. Such elements do not cause a line break. However, we can give height and width to these elements like how we treat the block level elements. Basically, "Display: Inline-Block" is a combination of both "Display: Inline" and "Display: Block" properties.
